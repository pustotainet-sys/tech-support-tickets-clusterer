### Описание скриптов в support-analyzer

## 1. analyze_support_requests.py — TF-IDF + KMeans
Метод: Классический подход на основе частотных признаков

Пайплайн:

Загрузка данных из Excel (DataExport_support22032026.xlsx)
Очистка текста (удаление HTML, URL, email)
TF-IDF векторизация — преобразование текста в числовые векторы на основе частоты слов (макс. 5000 признаков, биграммы)
KMeans кластеризация — 12 кластеров
Именование кластеров через YandexGPT (использует конфиг из kaiten-chat-rag-2)
Сохранение результатов в Excel
Зависимости: Требует основной проект kaiten-chat-rag-2 для доступа к YandexGPT

## 2. analyze_embeddings.py — Embeddings + KMeans
Метод: Современный подход на основе нейросетевых эмбеддингов

Пайплайн:

Загрузка данных из Excel
Очистка текста
Cloud.ru Embeddings — получение семантических векторов через модель BAAI/bge-m3 (1024 измерения) с кэшированием
UMAP — снижение размерности (1024 → 50)
KMeans кластеризация — 10 кластеров (без шума)
Именование кластеров через YandexGPT (автономный клиент)
Сохранение результатов в Excel
Зависимости: Автономный, требует только API-ключи в .env
